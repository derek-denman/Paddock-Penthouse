# Strategy Token Economy (MVP)

## Weekly Allocation
- Base tokens per event: `5`
- Carryover: none (resets each event)

## Costs
| Action | Token Cost |
|---|---:|
| Standard prediction | 1 |
| High-volatility prediction (`UNDERCUT`, `RESTART_LEADER`) | 2 |

## Rewards
- Correct standard prediction: +2 score points
- Correct high-volatility prediction: +4 score points
- Perfect 3+ streak in one event: +1 bonus token (usable same event)

## Insurance
- One miss protection per event if player has strategist level >= 2 (future staff feature).
